recoil로 말씀드리겠습니다.

recoil은 페이스북에서 만든 리액트 전용 상태관리 라이브러리입니다.
recoil은 
1. 비동기 처리를 기반으로 작성되어 동시성 모드를 제공하기 때문에 
thunk를 추가로 설치하여 사용하여야 하는 redux보다 라이브러리 자체 무게가 가볍습니다.

2. store라는 하나의 저장소가 아닌 더 작은 저장소 개념의 atoms들을 구독합니다. 

3. atom과 selector만 알아도 어느정도 구현이 가능하기 때문에 효율이 좋습니다.

4. store와 같은 외부 요인이 아닌 React 내부의 상태를 활용하고 
context API를 통해 구현되어있기 때문에 더 리액트에 가까운 라이브러리라고 할 수 있습니다.
