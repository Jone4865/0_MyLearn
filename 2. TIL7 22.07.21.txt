## **spread operator (전개 구문)**

전개 구문이란?**

기존에 있는 객체나 배열의 내용을 그대로 가져오거나 새로운 값을 덧붙여서 
새로운 객체나 배열을 생성 할 수 있습니다. 
기존의 데이터 (객체나 배열)에는 전혀 영향을 주지 않고 복사해서 새로운 것을 만듭니다.

const box = {size: "big", color: "red"};

const newBox = { ...box}; // {size: "big", color: "red"}
const newBlueBox = {...box, color: "blue" }; // {size: "big", color: "blue"}
